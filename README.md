# 2D Platformer Game

## Introduction
This project is a 2D platformer game built using Unity. It includes character movement mechanics, multiple levels, enemy AI, a scoring system, and UI elements.

## Prerequisites
- Unity 2021.3 or later
- Basic understanding of C# and Unity

## Setup
1. Open Unity and create a new 2D project.
2. Import the contents of this project into the new Unity project.
3. Open the `MainMenu` scene to start the game.

## How to Play
- Use arrow keys or `A` and `D` keys to move the character left and right.
- Press the `Space` bar to jump.
- Collect items to increase your score.
- Avoid enemies or defeat them by jumping on them.

## Build and Run
1. Open the `MainMenu` scene.
2. Go to `File` > `Build Settings`.
3. Add all scenes to the build.
4. Select your target platform and click `Build and Run`.
